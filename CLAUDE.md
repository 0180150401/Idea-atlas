# Idea Atlas Project Guide

This project uses GSD planning artifacts in `.planning/`.

## Current Project

Idea Atlas is a personal liminal atlas for ideas, aphorisms, notes, quotes, and intellectual fragments. It turns raw reading material and personal reflections into structured thought-objects, maps their semantic and philosophical relationships, and uses that atlas to generate tension-bearing aphorism bundles with provenance.

## Core Value

The system must help the user transform a corpus of fragments into original, provenance-aware idea constellations and aphorisms that preserve intellectual tension rather than flattening it.

## Read First

- `.planning/PROJECT.md` — living project context and decisions
- `.planning/REQUIREMENTS.md` — v1 requirements and traceability
- `.planning/ROADMAP.md` — phase structure and success criteria
- `.planning/STATE.md` — current phase and progress
- `.planning/research/SUMMARY.md` — synthesized research constraints

## Workflow

Use `/gsd:discuss-phase 1` next. After discussion, use `/gsd:plan-phase 1` to create the executable plan for Phase 1.

## Project Invariants

- Keep original source wording separate from extracted metadata.
- Preserve provenance for every source fragment and generated bundle.
- Model rhetorical form and worldview coordinates as first-class data.
- Prefer a working vertical loop over broad infrastructure.
- Treat generated aphorisms as bundles with lineage, reversal, and critique, not isolated one-liners.
